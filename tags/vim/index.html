vim中使用nerdcomment来注释python代码tips
intro nerdcomment是一个vim插件，提供多种文件类型的代码注释功能. 同时设置了写常用的快捷按键。 项目地址: https://github.com/scrooloose/nerdcommenter.git 常用的按键有: .. csv-table:: nerdcommnet按键 :header: keymap, 描述, key-mode :class: table :name: csv-table &lt;leader&gt;c&lt;space&gt;, 切换代码注释状态, v &lt;leader&gt;cc, 注释代码, v &lt;leader&gt;cl, 注释代码.但是保持左边对齐, v &lt;leader&gt;cb, 同上both-side.暂时没看出区别, v &lt;leader&gt;ca, 切换alt注释符号, n &lt;leader&gt;cA, 在行尾添加注释, n &lt;leader&gt;c$, 从当前cursor注释到行尾, n 配合python使用tips 添加空格 设置 `let g:NERDSpaceDelims = 1`. 之后注释的时候会在#后加入一个空格，这样代码做pep8的时候就不会报警告了. 如图. 设置前: .. image:: /static/vim-nerdcomment/bad-delims.jpg :alt: bad-pep8 设置后: .. image:: /static/vim-nerdcomment/good-delims.jpg :alt: good-pep8 注释对齐 使用 &lt;leader&gt;cl 取代 &lt;leader&gt;c&lt;space&gt;. 我一般也是用 &lt;leader&gt;c&lt;space&gt; 。但是注释出来的效果不是很好 看了好久文档总算是找个了解决办法.效果如下. c.. image:: /static/vim-nerdcomment/bad-align.jpg :alt: bad-pep8
